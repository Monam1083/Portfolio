## ✨ Website Features

This portfolio is built with pure HTML, CSS, and Vanilla JavaScript, prioritizing performance and bespoke animations without the overhead of heavy frameworks.

- **Custom Interactive Cursor:** A dynamic cursor dot and ring that expands when hovering over interactive elements.
- **GSAP-style Scroll Animations:** Staggered, smooth scroll reveals for the timeline, certification slides, and project grids using the Intersection Observer API.
- **Dynamic Loading Screen:** A sleek, mathematical percentage loader that executes before unveiling the DOM.
- **Infinite CSS Marquee:** A seamless, looping text ticker showcasing the core tech stack.
- **Live Contact Form:** Fully functional form integrated via Formspree API for instant email delivery.

## 🛠️ Built With

* **HTML5** - Semantic structure and SEO-optimized elements.
* **CSS3** - Custom properties, flexbox/grid layouts, keyframe animations, and responsive media queries.
* **Vanilla JavaScript** - DOM manipulation, custom cursor logic, intersection observers, and asynchronous form handling.
* **Formspree** - Backend form submission handling.
* **DevIcons** - Scalable SVG technology icons.
